# 슬랙 이모지용 슬라이딩 텍스트 GIF 생성기

Slack에서 사용할 수 있는 슬라이딩 텍스트 GIF 이모지를 쉽게 만들 수 있는 웹 도구입니다.

## 기능

- 텍스트 입력 및 커스터마이징
- 배경 색상 설정 (단색/그라디언트)
- 텍스트 색상 설정
- 슬라이딩 속도 조절
- 128x128 크기의 GIF 생성
- 투명 배경 지원

## 개발 환경 설정

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 기술 스택

- React
- TypeScript
- Tailwind CSS
- gif.js
- Vite
